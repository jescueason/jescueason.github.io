Push.js-Javascript桌面通知库

Push.js是一个可以快速运行和接收的Javascript桌面通知库，它支持IE9+，以及现代浏览器如：Chrome, Safari, Firefox。它可以应用在网页消息推送方面，而且简单易用。 
Push.create("Hello world!", {
    body: "恭喜您中奖了！",
    icon: 'images/icon.png',
    timeout: 4000,
    onClick: function () {
        window.focus();
        this.close();
    }
});
